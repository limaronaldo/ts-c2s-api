## Limitações da Pesquisa

1. **Dados Pessoais Protegidos (LGPD)**
   - CPFs são dados sensíveis protegidos pela Lei Geral de Proteção de Dados
   - Maioria dos CPFs não está disponível publicamente

2. **Work API**
   - Requer telefone para descoberta de CPF (módulo `phone`)
   - Módulo `cpf` requer CPF conhecido como entrada
   - CPF encontrado em SEC filing não retornou dados (pode estar incorreto)

3. **DuckDB API (223M CPFs)**
   - Buscas por nome são extremamente lentas (full scan)
   - Timeout após 10-15 segundos sem resposta

4. **Fontes Públicas Consultadas**
   - SEC filings (EUA)
